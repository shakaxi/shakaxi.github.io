1 Introduction

Today, I found an interesting \(\LaTeX\) package, gantt1. This package is developped based on pgf/tikz and, as its name indicates, can produce Gantt charts.

2 Installation

For ArchLinux, the package can be simply installed via yaourt. 

yaourt texlive-gantt

Alternatively, the corresponding style file can be downloaded from the webpage.

3 Usage

The framework of using the package can be illustrated as

3.1 Optional arguments

key_1, key_2, ... are optional arguments, which can be

xunitlength
length of one time slot (default: 1 cm)
fontsize
fontsize of labels (default: \normalsize)
titlefontsize
fontsize of title section (default: \small)
drawledgerline
switch to enable/disable the drawing of horizontal ledger lines (default: false)

3.2 Title

ganttitle is the environment for drawing the title section, which is comprised of a series of \titleelement and/or \numtitle

3.3 Other commands

3.3.1 \ganttbar

Draws a single, unconnected bar for a task. where

  • <pattern> is a tikz pattern, e.g. north west lines (default), crosshatch, crosshatch dots, grid, etc.
  • <color> is a tikz color of the pattern, e.g. black (default), red, green, blue, gray, dark gray.

3.3.2 \ganttcon

Draw an arrow between two bars with specified coordinates.

3.3.3 \ganttbarcon

Draw a single bar and connect it to the previous bar/milestone. where the optional pattern argument are the same as \ganttbar.

3.3.4 \ganttgroup

Draw a bar to group tasks.

3.3.5 \ganttmilestone

Draw a milestone using a diamond.

3.3.6 \ganttmilestonecon

Draw a milestone using a diamond and connect it to the previous bar/milestone.
